## Purpose
Gulpを用いて画像を一括で圧縮します。  
品質を損ねないように設定しています。  
Webサイト軽量化に使用しています。

### Supported image formats
- jpg / jpeg（クオリティ：90%）
- png（クオリティ：80％～95％）
- svg（SVGOで最適化）
 
### Required npm packages
- gulp
- gulp-cli
- del
- gulp-imagemin
- imagemin-pngquant

---
## How to Use
```bash
npm install
npm run build
```
